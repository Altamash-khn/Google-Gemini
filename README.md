# Google Gemini Clone

A sleek, user-friendly AI chatbot web app built with the **Gemini API** by Google. This project mimics the experience of the original Gemini interface while allowing you to integrate generative AI features like conversation, content generation, and more.

---

## Features

- Ask questions and get intelligent answers
- Chat UI similar to Gemini
- Handles loading and error states
- API key is stored securely using environment variables

---

## Tech Stack

- **React.js** – For building the user interface
- **Vite** – For fast development and build setup
- **Gemini API** – For generating AI-powered responses
- **Tailwind CSS** – For styling the app quickly using utility classes
- **Axios / Fetch** – For making API requests to Gemini
- **Netlify / Vercel** – For deploying the app online
