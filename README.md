# Google Gemini Clone

A modern, intelligent, and responsive **AI chatbot web app** inspired by **Google Gemini** — built with **React 19**, **@google/genai**, and **Vite**.  
This project replicates Gemini’s elegant interface and conversational flow, providing smooth animations, context-aware responses, and real-time chat rendering.

---

## Live Demo

[View Live Project](https://google-gemini-flame-gamma.vercel.app/)

---

## Features

- **Conversational AI Interface** — Ask questions and get smart, context-aware responses
- **Powered by @google/genai** — Uses the official Google Generative AI SDK
- **Gemini-style UI** — Clean and modern chat layout inspired by Google’s Gemini app
- **Smooth Animations** — Subtle motion and fade effects for a fluid user experience
- **Context API Integration** — Manages chat history and user session efficiently
- **Loading & Error States** — Interactive feedback for better UX

---

## Tech Stack

- **React.js** – For building the user interface
- **Vite** – For fast development and build setup
- **@google/genai** – Google’s Generative AI SDK for Gemini responses
- **Context API** – Handles chat state and data flow
- **CSS** – Custom styling for layout and responsiveness

## Installation & Setup

Follow these steps to run the project locally:

1. **Clone the repository**

   ```bash
   git clone https://github.com/Altamash-khn/Google-Gemini.git
   cd Google-Gemini
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Add your Gemini API Key**

   - add you gemini api key in the gemini.js file.

4. **Run the development server**

   ```bash
   npm run dev
   ```

5. **Open in your browser**
   ```
   http://localhost:5173
   ```
